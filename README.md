# Texto Giratório

Site com animação de texto giratório em 3D.

## Acesse o site
https://joseluissilvaferraz.github.io/texto-giratorio

## Tecnologias
- HTML
- CSS
- JavaScript
- dat.GUI

## Como usar
- Abra o site
- Use o painel no canto superior direito para personalizar:
  - **Font Size** → tamanho da fonte
  - **Height** → altura da sombra
  - **Reflection Opacity** → opacidade do reflexo
  - **Reflection Blur** → desfoque do reflexo
  - **Text** → mude o texto giratório
